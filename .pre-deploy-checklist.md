# 배포 전 체크리스트

배포하기 전에 다음 항목들을 확인하세요.

## ✅ 필수 확인사항

- [ ] `npm run build` 명령어가 성공적으로 실행되는지 확인
- [ ] `npm run preview`로 빌드된 파일이 정상 작동하는지 확인
- [ ] 모든 기능이 정상 작동하는지 테스트:
  - [ ] 링크 입력
  - [ ] 이미지 업로드
  - [ ] QR코드 미리보기
  - [ ] QR코드 생성 및 다운로드
  - [ ] 다중 항목 추가/삭제
- [ ] 모바일에서도 정상 작동하는지 확인
- [ ] `.gitignore`에 `node_modules`와 `dist`가 포함되어 있는지 확인

## 📝 배포 전 테스트

```bash
# 1. 빌드 테스트
npm run build

# 2. 로컬에서 빌드된 파일 테스트
npm run preview

# 3. 브라우저에서 http://localhost:4173 접속하여 테스트
```

## 🚀 배포 단계

### Vercel 배포 (가장 쉬움)

1. GitHub에 코드 푸시
2. [vercel.com](https://vercel.com) 접속
3. GitHub 저장소 연결
4. 자동 배포 완료!

### Netlify 배포

1. GitHub에 코드 푸시
2. [netlify.com](https://www.netlify.com) 접속
3. GitHub 저장소 연결
4. 빌드 설정 확인 후 배포

## 🔍 배포 후 확인

- [ ] 배포된 사이트 접속 확인
- [ ] 모든 기능 테스트
- [ ] 모바일에서 접속 테스트
- [ ] QR코드 생성 및 다운로드 테스트
- [ ] 페이지 새로고침 시 정상 작동 확인

## ⚠️ 문제 발생 시

1. 브라우저 콘솔에서 에러 확인 (F12)
2. 빌드 로그 확인
3. `npm run build` 로컬에서 다시 테스트
4. Vercel/Netlify 대시보드에서 빌드 로그 확인
